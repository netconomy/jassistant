===============
Sprint analysis
===============

Sprint Analysis is a module to analyse sprints, what was finished etc.. Multi
Sprint Analysis gives you the option to chain multiple Sprint Analysis and get
them in a single CSV file.

If you have trouble finding out what ID your sprint has look it up the Sprint
Id documentation below.
